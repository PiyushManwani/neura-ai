\# 🧠 Neura AI



A beautiful terminal-based AI assistant powered by Ollama. Chat with your own custom AI model that runs \*\*100% locally\*\* on your PC.



!\[Banner](https://via.placeholder.com/800x200?text=Neura+AI+Screenshot)



\## ✨ Features



\- 🎨 \*\*Beautiful TUI\*\* — Claude Code-inspired interface with dark theme

\- 🧠 \*\*Your own AI\*\* — Custom `neura` model with personality you control

\- 🦙 \*\*Powered by Ollama\*\* — Free, local, private

\- 📄 \*\*PDF Import/Export\*\* — Read PDFs into chats, save chats as PDFs

\- ⚡ \*\*Fast streaming\*\* — Real-time token streaming

\- 🎯 \*\*30+ commands\*\* — Models, settings, personas, and more

\- 🔒 \*\*100% Private\*\* — Everything runs on your machine



\## 🚀 Installation



\### Windows (One Click!)



```cmd

git clone https://github.com/PiyushManwani/neura-ai.git

cd neura-ai

install.bat

```



\### Mac/Linux



```bash

git clone https://github.com/PiyushManwani/neura-ai.git

cd neura-ai

chmod +x install.sh

./install.sh

```



\## 📋 Requirements



\- \*\*Windows 10/11\*\*, macOS, or Linux

\- \*\*8GB RAM minimum\*\* (16GB recommended)

\- \*\*500MB disk space\*\* for the small model

\- \*\*Python 3.10+\*\* (installer can fetch it)



\## 🎯 Usage



After installation, just type:



```bash

neura

```



\### Common Commands



| Command | Action |

|---------|--------|

| `/help` | Show all commands |

| `/models` | Pick from available models |

| `/model neura` | Switch to your custom AI |

| `/import <pdf>` | Read a PDF into the chat |

| `/export` | Save chat as styled PDF |

| `/persona coder` | Use a preset personality |

| `/quit` | Exit |



\## 🎨 Customize Your AI



Edit `Neura.Modelfile` to change Neura's personality:



```

SYSTEM You are Neura, a sarcastic coding mentor who loves Python and Rust.

```



Then re-run `install.bat` to rebuild.



\## 🗑️ Uninstall



```bash

uninstall.bat   # Windows

```
